# KED - LED: Andrew's proposal

•	Press and hold >5s to enter maintenance mode
o	Keep behavior of entering this mode on button release and designate with rapid flashing for some seconds
•	Once in selection mode and rapid flashing is finished, existing fieldbus type begins its LED behavior
•	A single button presses flip to the next fieldbus LED behavior
•	Press and hold >5s to lock in current selection and exit maintenance mode
o	Again, the selection is saved on release of the button and designated by rapid flashing for some seconds
o	Worst case, if button slips before 5s, user has only traversed to next option and must flip back through, which doesn’t seem too bad
•	After maintenance mode exit, drive returns to normal LED behavior
